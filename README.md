## About
My personal configuration files for i3wm

## Requirements
`i3` \
`feh` \
`ImageMagick` (only for lock screen)

## Usage
Copy configuration files and reload i3: \
`xrdb ~/.Xresources` \
`super` + `shift` + `r`

The usual place for config files
- `~/.config/i3/config` (i3wm)
- `~/.Xresources`
- `~/.config/i3status/config` (i3status)
